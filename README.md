# taller1_microapi
# Sistema de Registro de Estudiantes - API REST

API RESTful desarrollada con Spring Boot para el registro y consulta de estudiantes. Los datos se almacenan en memoria durante la ejecución del servidor.

# Tecnología Utilizada

Java 17 + Spring Boot 3.2 + Maven**

# Requisitos Previos

- Java 17 o superior
- Maven 3.6 o superior

# Instalación y Ejecución

### 1. Clonar el repositorio
```bash
git clone 
cd student-api
```

### 2. Compilar el proyecto
```bash
mvn clean install
```

### 3. Ejecutar la aplicación
```bash
mvn spring-boot:run
```

O alternativamente:
```bash
java -jar target/student-api-1.0.0.jar
```

El servidor se ejecutará en: `http://localhost:8080`
